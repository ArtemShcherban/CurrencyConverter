An app for a quick assessment of current exchange rates. CurrencyConverter allows users to view exchange rates from any day of the past year and share rates through messengers and more. Adapted for different device (iPhone / iPad) and orientations (portrait / landscape).

UIKit – MVVM – CoreData – RESTful API – Auto Layout (fully storyboard) – Core Animation 
– URLSession – Combine – UnitTests – UITests – Git – SwiftLint
